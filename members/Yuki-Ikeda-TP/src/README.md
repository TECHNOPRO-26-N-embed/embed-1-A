# ソースコードフォルダ

実習のソースコード（.c / .ino ファイル等）をここに置いてください。

## この課題の実装ファイル

```
password_led_judge.ino
```

## 依存ライブラリ

追加ライブラリは不要です。

## 動作確認手順（Arduino IDE）

1. `password_led_judge.ino` を開く
2. ボードを `Arduino Uno`、ポートを実機に合わせて選択する
3. コンパイルして書き込む
4. シリアルモニタを 9600bps で開く
5. キーパッドで4桁入力後、`#` で判定する

## ファイル例

```
unit1_hello.c
unit2_led.ino
```
