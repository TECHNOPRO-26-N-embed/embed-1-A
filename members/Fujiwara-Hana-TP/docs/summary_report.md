【発表原稿（2分想定）】

私は「プロペラの風量を選択できるボタン」というガジェットを作りました。
（ここで実物を見せる）

この装置は、キーパッドを使って、ファンの回転速度を0から9段階で自由に選べるのが特徴です。従来の扇風機は「弱・中・強」の3段階で、細かい調整ができませんでしたが、このガジェットなら自分の好みに合わせて細かく風量を設定できます。

（実物を見せながら）
使い方は、キーパッドで数字を押すだけで、その段階の風量にすぐ切り替わります。0を押すとファンが止まります。今どの段階かはシリアルモニタにも表示されるので、動作確認も簡単です。

設計で工夫した点は、誰でも直感的に操作できるようにしたことと、無効な入力があった場合はファンを自動で停止し、エラーを表示する安全設計にしたことです。

一番苦労したのは配線です。最初はどこにどの線をつなぐか混乱しましたが、Arduino UNOのリファレンスやAIのアドバイスを活用し、1本ずつ丁寧に確認していくことで、すべての部品が正常に動作するようになりました。

今回の制作を通じて、AIを使うときは「何を参照し、何をしたいか」を明確にプロンプトに書くことが大切だと学びました。
