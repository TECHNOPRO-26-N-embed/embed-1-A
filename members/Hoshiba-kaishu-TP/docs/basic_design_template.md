## 1. システム全体像

### 1-1. ブロック構成（入力 → 処理 → 出力）

```
【入力】                 【Arduino UNO R3】            【出力】
タクトスイッチ ─────→┐                         ┌──→ モーター
                      ├── 状態管理・判定処理 ──┤
                      │   ・パスワード認証      │
                      │   ・速度/タイマー設定    │
                      │   ・安全監視            │
                      └─────────────────────────┘
                                 │
                                 └──→ LED
```

#### 補足説明
- 入力：ユーザーは複数のタクトスイッチでパスワード・速度・タイマーを設定。
- 処理：Arduinoが状態遷移・安全判定・タイマー管理を行う。
- 出力：モーター制御、LED表示、ブザー警告。

---

### 1-2. 状態遷移（ガジェットの「ふるまい」設計）

```
[電源ON / 初期化]
  ↓（初期化完了）
[待機中] ──（パスワード認証成功）──→ [動作中] ──（タイマー終了/手動OFF）──→ [停止中]
   ↑                                                            │
   └────────────────────（リセットボタン）────────────────────────┘
```

| 状態名 | この状態のとき、何をしているか | LEDやブザーなど出力の様子 |
|:--|:--|:--|
| 待機中 | パスワード入力待機。速度/タイマー設定不可。 | LED点滅（緑） |
| 動作中 | モーター回転、速度・タイマー調整可。 | LED点灯（青）、残り時間点滅（任意） |
| 停止中 | モーター停止。再認証で再始動可。 | LED消灯、必要に応じてブザー |
| エラー | 異常検知時（例：誤パスワード3回、過電流） | LED点滅（赤）、ブザー連続音 |

#### 状態遷移の詳細
- 誤パスワード3回で30秒ロック→自動で待機中に戻る。
- タイマー終了時は自動で停止中へ。
- 異常時はエラー状態に遷移し、ユーザー操作で復帰。

### 2-3. ユーザーインターフェース設計

- **LED表示**
  - 緑点滅：待機中
  - 青点灯：動作中
  - 赤点滅：エラー
- **ブザー**
  - 誤パスワード時：短いビープ
  - 異常時：連続ビープ
- **ボタン操作例**
  - A：パスワード入力モード
  - B：速度設定モード
  - C：タイマー設定モード
  - D：電源ON/OFF
  - 0-9, *, #：数値入力・確定

---

### 2-4. 運用シナリオ例

1. 電源ON → LED緑点滅
2. Aボタンでパスワード入力モード → 4桁入力し#で確定
3. 正しければLED青点灯・速度/タイマー設定可
4. B/Cボタンで速度・タイマー設定、Dで運転開始
5. タイマー終了またはDで停止 → LED消灯
6. 誤パスワード3回で赤点滅・30秒ロック

## 4. 異常系・エラー処理の方針

| 異常ケース | 発生条件 | 検知方法 | 対応方針 |
|:--|:--|:--|:--|
| チャタリング | ボタンを押したとき | 前回入力から50ms未満の入力を検出 | デバウンス処理でスキップ |
| 誤パスワード | パスワードが一致しない | 入力値と正解値を比較 | 再入力を促す・3回でロック、ブザー短音・LED赤点滅 |
| タイマー異常 | 設定値が範囲外 | 入力値チェック | 最小/最大値に補正・LED赤1回点滅で通知 |
| モーター異常 | 過電流・停止検知 | 電流センサ等（任意） | 自動停止・ブザー連続音・LED赤点滅 |
| 間違った操作 | 状態に合わないボタン押下 | 状態と入力の組合せを判定 | 無効操作は無視し、LED赤1回点滅またはブザー短音で警告 |

### エラー・間違い操作時のユーザー通知例
- パスワード未認証時に速度/タイマー/電源ボタンを押した場合：LED赤1回点滅＋ブザー短音（操作無効を通知）
- タイマー設定範囲外の値を入力した場合：LED赤1回点滅＋ブザー短音（自動で最小/最大値に補正）
- 誤パスワード3回：LED赤高速点滅＋ブザー連続音（30秒ロック）
- モーター異常時：LED赤点滅＋ブザー連続音（自動停止）
- その他無効な操作：LED赤1回点滅またはブザー短音

### 復帰方法
- ロック解除やエラー復帰後はLED緑点滅に戻り、再度パスワード入力待機状態へ
- エラー状態でリセットボタンを押すと初期状態に戻る
