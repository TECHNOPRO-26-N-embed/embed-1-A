## 4. 一番苦労したこと、どう乗り越えたか（30秒）

<!-- ここが発表の山場です。1つだけに絞って、ストーリーで書いてください -->

**何が起きたか：**
配線が一番の苦労しました。
キーパッド・モーター・LEDはそれぞれ単体なら動かせたのですが、3つを同時に組み合わせると配線が複雑になり、調整に苦労しました。

**どう対処したか：**
まずは一つ一つどう動くかを確認して最初は２つ同時に動くかを確認し、違うパターンで２つ同時に動くかを確認してから３つを統合しました。

**そこから何がわかったか：**
今回わかったことは、複数の部品を組み合わせる際は、個別に動作確認を行いながら段階的に統合することが重要だと実感しました。また、配線図や設計図を事前にしっかり描いておくことで、トラブルを未然に防げることがわかりました。
